1. 投资哲学：我是谁，我赌什么

核心信念 —— "我不知道未来是什么样，但我知道未来需要什么。买未来必须依赖的东西。"

1.1 我的身份认知

  • IT/互联网行业从业30年，AI/半导体/硬件/软件资深专家
  • 能力圈：技术路径判断、产业链结构、商业化时间窗口、中美产业差异
  • 不是巴菲特(消费品+品牌)，不是段永平(看人+生意模式)，更接近 John Malone(管道+通行权+垄断地段) 和 19世纪的卡内基/洛克菲勒

1.2 我赌的是什么

层级我赌我不赌
产业方向AI是10-30年的基础设施革命(类比电力/铁路)不赌"AI 3年内见顶"或"灭世级颠覆即将发生"
赢家不管谁赢，必须用谁的"地盘"不赌"哪家应用层公司会赢"
定价资本市场把10年长周期错误压缩成3年短周期，给了基础设施折价不赌"市场永远理性"
颠覆真颠覆只有两种：量子计算(2035-2050) + 神经网络数学被改写(2030-2045)不为这两个尾部风险持续付溢价

1.3 三条铁律(认知锚)

铁律A 中美算力隔离 —— 中国缺算力，美国缺开源模型。GLM-5.2级别的开源模型超过Claude 4.6 Opus只触发中国市场token商品化，不影响北美定价。任何"全球化AI叙事"都要先按这条铁律切两半。
铁律B 稀缺性分级(分语境) —— 全球语境下：物理稀缺 > 软件壁垒 > 政策稀缺(关系缓和就打折)。但中国语境下：物理稀缺 ≈ 政策稀缺 > 软件壁垒——中美博弈把出口管制+国产替代闭环铸成了"类物理壁垒"，5年内不可逆。CUDA这种软件生态在两个语境里都最弱，10年内可能被开源绕过。决策时先判断标的所处语境，再按对应等级付溢价。
铁律C 产业计量30年，持仓计量3-5年 —— 基础设施周期(铁路80年/电力80年/电信90年/移动30年/互联网骨干25年)用30年衡量产业方向是否还在；但我的持仓周期是3-5年一轮，跟着capex周期/估值钟摆/技术节点跃迁换股。"30年"防止我被一年期的市场噪音吓出场，"3-5年"防止我把节点资产变成压在床底的死钱——产业方向不变，但谁在收过路费会换人。

2. 能力圈定义：我懂什么，不懂什么

2.1 能力圈内（我应该比绝大多数人判断更准）

强能力圈
  • AI技术路径：训练 vs 推理、ASIC vs GPU、开源 vs 闭源、模型架构演进
  • 半导体产业链：制程节点、代工vs IDM、HBM/CoWoS瓶颈、设备国产化进度
  • 互联网商业化：流量、SaaS、广告、云、电商
  • 中美产业差异：算力分布、模型生态、应用层节奏、政策影响
  • 商业化时间窗口判断：哪些技术12-18个月落地，哪些5-10年

2.2 能力圈外（我不该用技术框架推理）

错配警告
  • 消费品/品牌/IP：人造稀缺、心智占领、文化生命周期 → 巴菲特/段永平的领域
  • 医药/CXO/创新药：FDA合规、跨国客户关系网、毒理学家团队留存、专利+流程工程
  • 金融/保险/地产：监管套利、资本充足率、利率敏感度、杠杆周期
  • 大宗商品/资源股：供需周期、库存周期、地缘溢价

对这些领域：用技术方法论推理 = 错配，必须改用对应领域的专业框架，或承认不投，或降级为"小赌注 + 学习仓"。

2.3 学习仓机制

消费/医药这种"周期性但永续"赛道(人不会停止吃喝玩乐、不会停止生病)，应建立长期学习仓：

  • 仓位上限：单赛道总仓位 ≤ 5%，所有学习仓加起来 ≤ 10%
  • 目的：不是赚钱，是为2030年代AI热度退潮时储备能力圈
  • 方法：每月固定金额买，强迫自己看年报、跟踪行业、学习估值框架
  • 禁忌：不许用AI/科技逻辑给消费股估值，不许把"AI+医疗最大市场"当成重仓理由

3. 仓位结构：分裂的伯克希尔

总仓位 100% ├── 节点压舱石 30%(牛市) → 50%(常态) → 70%(熊市) │ ├── 全球算力节点 (海外/物理稀缺主导) │ ├── 中国算力节点 (政策稀缺 ≈ 物理稀缺,中美博弈期) │ └── 长期赛道学习仓 ≤ 10% (消费/医药) │ └── AI alpha 70%(牛市) → 50%(常态) → 30%(熊市) ├── AI铲子β仓 (短期，赚泡沫快钱) ├── AI native应用 (你的真α) ├── 强制丑陋仓 ≥ alpha仓的 15% (心理对冲) └── 短线消息面 ≤ 5% (硬上限)

3.1 节点压舱石(替代债券+黄金)

定位 —— 这不是"风险资产"，是"高质量节点 = 可承受波动的债券"。它替代债券+黄金的功能，但收益率高3倍。
  • 下行特征：99.99%不会归零(物理资产+全球客户+刚需服务)，最大回撤可能-50%但永远恢复
  • 上行特征：年化8-15%(产业基础设施租金+复利)，不会10倍但稳健
  • 持仓周期：3-5年一轮(对应capex周期/估值钟摆/技术节点跃迁)。每一轮内根据PE÷均值比值在标的之间换仓；轮次结束时全面重估"谁还是节点"——产业方向不变，但收过路费的人会换
  • 单只上限：不超过总仓位的30%，防"我太确信"
  • 禁忌：永远不加杠杆；轮次内不止盈(节点物理被替代除外)；但轮次切换点必须重新审视，不允许"放进去就忘记"

3.2 AI alpha仓(我的能力圈复利)

  • 定位：用30年从业经验在我熟悉的领域赚α，不是赌而是套利
  • 核心套利：技术兑现快的公司 + AI native早期赢家 + 中美算力隔离的错位定价
  • 泡沫β仓的必要性：AI铲子虽然low，但泡沫维持期能赚快钱。设硬时间窗(如"半年内退出")，到期不论盈亏强制砍掉
  • 强制丑陋仓 ≥ 15%：必须永远持有"我看不上但市场在涨"的标的(NVDA类)，对冲审美偏见
  • 短线消息面 ≤ 5%：禁止追消息面短线超过这个比例，防止情绪化操作侵蚀alpha本金

3.3 牛熊比例切换规则

市场状态压舱石占比alpha仓占比触发条件
极端熊市70%30%AI ETF -30%、VIX>40、CSP capex连续2季下调>15%
常态50%50%无极端信号
明显牛市30%70%美股/A股估值分位>70%、AI泡沫温度计>75
反直觉但正确：三七开 / 五五开 / 七三开 —— 大部分人牛市加杠杆、熊市减仓，是顺周期。我反过来：牛市才是赚alpha的窗口期(铲子股泡沫维持、估值钟摆向上、技术兑现被放大定价)，所以alpha升到70%；熊市压舱石才是真本钱(节点资产被打折、便宜买入物理稀缺的入场点)，所以压舱石升到70%。压舱石的本质是防御+逢低加，不是"牛市的主力"。

切换不是一次性，而是每月评估一次，每次最多调5%，避免择时风险。状态判定优先看温度计+估值分位，不看大盘指数本身。

4. 标的筛选：节点 vs alpha

4.1 节点资产准入清单(过门必须全YES)

检验项标准
① 物理稀缺度资产替代需要 ≥ $100亿资本+5年时间，或全球前3寡头
② 客户多元化前5大客户占比 < 60%，且分布跨行业/跨国
③ 商业模式收"过路费"/"租金"，而非赌"谁会赢"
④ 颠覆触发条件被替代需要量子计算成熟 OR 神经网络数学被改写 OR 政策极端反转
⑤ 历史回撤过去20年最大回撤 ≤ -60%且恢复时间 ≤ 5年
⑥ 现金流质量FCF/净利润 ≥ 60%，资本支出可预测
⑦ 节点等级纯垄断(独家) > 准垄断(份额70%+) > 政策稀缺。注：中美博弈期内政策稀缺 ≈ 准垄断；博弈缓和后立即降级

4.2 alpha资产筛选(我的能力圈套利)

  • 技术兑现型：12-18个月内有产品里程碑/客户突破/财务拐点的公司
  • 错位定价型：中美隔离造成的同质资产价差(如A/H溢价、美股vs港股的同业对比)
  • 早期赢家型：AI native应用公司，用我对产品形态的理解提前2-3季度下注
  • 禁区：完全凭"故事"无产品/无财务路径的concept股；纯模型公司(没有分发/没有数据壁垒)

4.3 拒绝清单(永远不碰)

禁投三类
  • 纯故事公司(财务无路径，靠PR+概念维系市值)
  • 能力圈外的高估值公司(用错框架推理 = 大概率亏钱)
  • "看起来便宜"但增速<0的Value Trap(便宜不是理由，便宜+加速才是理由)

5. 估值与决策框架

5.1 节点资产：PE÷均值PE 比值法

对成熟节点公司(增速15-30%)，唯一指标是"当前PE ÷ 历史均值PE"。比值最低 = 相对最便宜 = 持有谁。

比值状态动作
< 0.85绝对低估加仓 / 满仓
0.85 - 1.00低于均值持有 / 加仓
1.00 - 1.15均值附近持有 / 观望
1.15 - 1.30偏贵观望 / 不加
> 1.30明显高估等待 / 部分获利
均值锚的特殊处理 —— 历史均值不是简单算术平均：
  • 被打压的中概股：用"打压前均值 × (1 - 中概折价35-40%)"作为锚
  • 被低增速期拉低的科技股：用中位数而非均值
  • IPO时间短(<5年)：均值不可靠，用同行业对标

5.2 alpha资产：PEG + 期望值

  • 用Forward PE(机构一致预期)而非TTM
  • PEG < 0.5 = 强烈低估；0.5-1.0 = 合理；> 1.5 = 看故事
  • 三档建仓：A线30% / B线60% / 极端档100%
  • 期望值决策：胜率 × 上行空间 - (1-胜率) × 下行空间，正期望才动手

5.3 跨市场套利：价格归一化指数法

同一公司A/H/US多地上市时，把所有市场股价归一化到IPO日=100，看相对涨跌差距而非PS/PE。

  • 差距 ≤ 5pct：随便选(看流动性/税负)
  • 差距 5-15pct：倾向便宜市场
  • 差距 15-25pct：强制选便宜市场
  • 差距 > 25pct：买便宜市场+耐心等回归
  • 反向避坑：差距收敛通常是"贵市场补跌"而非"便宜市场补涨"，所以永远买便宜端

5.4 卖出触发条件

持仓类型触发卖出
节点压舱石仅在比值 > 1.30 时部分获利(不全卖)；物理被替代时清仓
AI alpha-技术兑现财务兑现 + 估值过度反应；或3年技术路径被证伪
AI alpha-早期赢家商业化路径明确后估值过度反应；或被巨头收购/替代
AI铲子β仓硬时间窗(预设半年/一年)，到期强制砍；或泡沫破灭信号触发
短线消息面消息兑现立即砍；连续2周无新催化砍掉

6. 心理纪律：两条护栏

6.1 护栏A：防过度自信 / 时间窗口误判

风险 —— "我太懂"会让我给某个判断加过高权重。即使判断方向正确，时间窗口可能比预期长2-3年，期间-40%回撤会触发情绪化操作。
  • 单一节点持仓上限30%：再确信也不能超过
  • 杠杆禁令：节点公司任何情况不加杠杆；alpha仓单笔杠杆 ≤ 50%
  • 2年延后压力测试：下注前必须算"如果对但晚兑现2年，期间最大回撤多少，能扛住吗?"
  • 时间戳判断：任何"X月内XX破灭"的判断必须打卡，到期必须复盘"为什么对/错"，再决定延期还是撤回

6.2 护栏B：防"low感"偏见 / 错过β

风险 —— 行业人的审美偏见会让我讨厌"看起来low"的赢家。NVDA股价表面看就是泡沫，但泡沫维持5年也能让人翻倍，因为"看不上"而错过最大的β。
  • 强制丑陋仓 ≥ AI alpha仓的15%：永远持有一定数量"看起来傻但市场在涨"的标的
  • 季度审美审计：每季问自己"过去3个月哪些股涨超50%但我没买? 真有问题还是审美在挑剔?"，存档
  • 悔不当初指数：记录所有"看上但没买"的标的，每月看一次。累计错过β > 当年alpha收益的30% → 偏见警报
  • 做空高门槛：节点公司可无视卖方研报，但bubble股必须看一致预期；盈利预测连续2季下调才允许做空

6.3 一般纪律(适用所有持仓)

  • 禁止追消息面短线超过5%：消息面是噪音不是信号
  • 不画"终局配置图"：只做今天能行动的判断，不做"5年后理想配置"的空想
  • 承认错了就改：连续2季基本面证伪 → 强制清仓，不许"再等等"
  • 赛道接力法：爆发期建龙头，增速<15%退出，换下一个赛道。没有股票要永远持有(节点压舱石除外)
  • 段永平方法论作为边界：他的方法论给纪律(不懂不投/不预测/平价持有/看人看模式)，但判断"懂不懂"是我自己的事——AI领域我比他懂，AI外参考他

  • 节点压舱石 PE/均值比值排名(ai-rotation.html)

7.2 周度复核

  • alpha仓位的兑现进度 vs 时间窗口
  • 短线消息面持仓清理 / 续期决策
  • 新增观察标的入观察区(zone='watch')

7.3 月度评估

  • 牛熊状态 / 仓位比例切换 (压舱石50-70% vs alpha 30-50%)
  • 悔不当初指数计算 → 偏见警报
  • 学习仓定投执行 + 学习笔记

7.4 季度审计

  • 所有"X月内XX破灭"判断的复盘
  • 审美审计：哪些涨50%+的我没买? 为什么?
  • 能力圈边界检视：是否有标的我用了错的框架?
  • 节点资产准入清单复核(7条标准是否仍满足)

7.5 年度迭代

  • 方法论本身的版本升级(本文档迭代到v2.0/v3.0)
  • 能力圈学习仓的"毕业"判断：消费/医药是否可以从学习仓升级到正式仓?
  • 段永平/巴菲特等参考体系的吸收 → 我的体系更新
方法论的元规则 —— 这份文档每年至少更新1次。任何重大决策错误后立即触发更新。方法论不是教条，是不断逼近真实的工具。
